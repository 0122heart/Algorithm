# Algorithm

요일별로 최소 한 문제는 풀기

| 요일   | 알고리즘 범위               |
|--------|-----------------------------|
| 월요일 | Dynamic Programming         |
| 화요일 | 자료구조 (스택, 힙, 덱, 리스트) |
| 수요일 | 탐색 & 그래프 (BFS, DFS, Binary Search 등) |
| 목요일 | 시뮬레이션 (구현 문제)      |
| 금요일 | 완전탐색 & 백트래킹         |
